## üé® Design Guidelines

### Consistent Elements Across All Infographics

**Color Palette**:
- Primary: Deep blue (#0B1929)
- Accent: Bright cyan (#00D4FF)
- Success: Green (#00C853)
- Warning: Orange (#FF6D00)
- Error: Red (#D50000)
- Neutral: Grays (#F5F5F5 to #212121)

**Typography**:
- Headers: Modern sans-serif (e.g., Inter, Helvetica)
- Body: Clean readable font
- Code/Technical: Monospace font
- Numbers: Tabular figures for data

**Visual Motifs**:
- Hexagonal patterns (blockchain/distributed systems)
- Circuit board traces (data flows)
- Lock icons (security/encryption)
- Node networks (decentralization)
- Gradient overlays (layer transitions)

**Animation Suggestions**:
- Data flowing through pipelines
- Nodes connecting/disconnecting
- Token distribution flows
- Real-time metric updates
- Layer build-up sequences

**Responsive Considerations**:
- Desktop: Full detailed views
- Tablet: Simplified but complete
- Mobile: Key metrics focus
- Print: High-contrast versions
